# 编译CSS
``` bash
npm run build:css
```

# 预览效果
``` bash
npm start
```
## 或者
``` bash
npm start -- --disable-gpu
```

# 打包
``` bash
npm run make
```

# 其它命令
1. 安装electron模块
 ``` bash
 ELECTRON_MIRROR="https://npmmirror.com/mirrors/electron/" npm install electron --save-dev --verbose
 ```
 2. 安装指定版模块
``` bash
 npm install tailwindcss@3.4.1 --save-dev
 ```
